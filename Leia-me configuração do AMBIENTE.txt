Caso não tenha o Python/VirtualEnv/Django instalado será necessario fazer isso antes dos comandos a frente.

1.1: python --version
1.2: python -m django --version
1.3: python -m django --version
Para Instalar no projeto, caso já exista o projeto vá para o passo 7.  
2: virtualenv venv
3: venv\Sripts\activate
4: pip install django
5: pip freeze
6: pip freeze > requirements.txt
------------------------------
7: Caso exista o projeto só execute o comando a frente: install -r requirements.txt
------------------------------

9: python manage.py runserver
